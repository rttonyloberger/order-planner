# Order Planner

Live hosted order management app for RT and SG purchasing operations.

## Setup

1. Run the SQL in `schema.sql` in your Supabase SQL Editor
2. Copy `.env.example` to `.env` and fill in your Supabase URL and anon key
3. Add those same values as Environment Variables in Vercel

## Tech Stack
- React 18
- Supabase (database + real-time)
- Vercel (hosting)

## Deployment
Push to GitHub main branch → Vercel auto-deploys.
